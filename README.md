# 🎮 Gerador de Famílias para The Sims 4

Um projeto criativo e divertido desenvolvido para gerar famílias aleatórias para o jogo **The Sims 4** 🏡✨

* [Acesse o projeto online](https://juliabaptistaalmeida.github.io/)

---

## 💡 Sobre o projeto

Este é um gerador de famílias que permite criar combinações únicas e inesperadas para enriquecer sua experiência no jogo.

O programa foi originalmente desenvolvido em **Python** e adaptado para a web utilizando **JavaScript, HTML5 e CSS**, proporcionando uma interface simples e acessível diretamente no navegador.

---

## ⚙️ Funcionalidades

Você pode gerar famílias com:

* 👨‍👩‍👧 Quantidade personalizada de Sims (de 1 a 8)
* 🧬 Gênero e idade definidos automaticamente
* 🧛‍♂️ Diferentes raças/criaturas, como:
  * Humanos;
  * Vampiros;
  * Sereias;
  * Gatos;
  * Cavalos;
  * E outros...
* 🎭 Traços de personalidade sorteados automaticamente

Cada geração cria uma família única, perfeita para novos desafios ou histórias dentro do jogo 🎲

---

## 🚀 Tecnologias utilizadas

* JavaScript
* HTML5
* CSS3
* Python (versão original do projeto)

---

## 🎯 Objetivo

O objetivo deste projeto é estimular a criatividade dos jogadores, ajudando a criar famílias variadas e interessantes sem precisar pensar manualmente em cada detalhe.
